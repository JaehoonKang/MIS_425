Instructions:

You MAY NOT alter the html or css files to complete this assignment. Points will be deducted if either the html or css files are changed. You may only work within the JavaScript (.js) file. Everything you need to complete this assignment can be found in chapters 8 & 9.

Part 1

Your objective is to output the values of the form fields into the Form Results section. The form has an id="myForm". The Form Results section contains a div tag with an id="results". You will display the resuts of the form there. Each form field should be displayed on its own line. For example:

First Name = Scott

Last Name = Geiger

Favorite Day = Fri

Pick Your Color = ltBlueBg

Part 2

After you have the results entered into the Form Results section, add code such that the background color of the section is whatever value is selected in the "Pick Your Color" list. Your css file contains classes that match the value of the select list options. Hint: you will need to remove any previously set class before adding a new one.

First Name = Scott

Last Name = Geiger

Favorite Day = Fri

Pick Your Color = ltBlueBg

Part 3

Change your code such that when the "Pick Your Color" select list changes the background color of the results section also changes. Note: You do NOT have to update/change the value listed in the results; only the background color needs to change.

Part 4

The output of the select lists (Favorite Day and Pick Your Color) are the values associated with the options. Change the output to match what is displayed in the select list. For example:

First Name = Scott

Last Name = Geiger

Favorite Day = Friday

Pick Your Color = Light Blue
